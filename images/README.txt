Para completar o cabeçalho do PDF:

1. Já temos o arquivo logo.png nesta pasta

2. Precisamos adicionar o ícone do WhatsApp:
   - Baixe um ícone do WhatsApp em formato PNG (preferencialmente com fundo transparente)
   - Salve como "whatsapp.png" nesta pasta
   - Tamanho recomendado: aproximadamente 64x64 pixels

O sistema irá:
- Usar o logo.png como logotipo principal
- Usar o whatsapp.png como ícone pequeno ao lado do número de telefone
- Se alguma imagem não for encontrada, o sistema continuará funcionando, apenas não mostrará a imagem faltante
